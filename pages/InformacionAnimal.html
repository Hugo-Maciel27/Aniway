Conejo

✓ El 80% de su alimentación debe ser heno, 15% verduras y 5% de pienso.
✓ Medida mínima de recinto 120cm x 60cm
✓ Mantener el bebedero con agua fresca.
✓ Evitar usar sustrato.
✓ Limpiar seguido el recinto.

Gato

✓ Cama lejos de las corrientes de aire y de fácil acceso.
✓ Ubique su arenero en un lugar tranquilo y al que pueda acceder con facilidad.
✓ Evite que se le formen las bolas de pelo con cepillados.
✓ Proporcionarle una herramienta que le permita arañar todo lo que quiera (un rascador).
✓ Es imprescindible que tenga tres recipientes, uno para la comida seca, otro para la comida húmeda y otro para el agua.

Aves

✓ Jaula grande donde pueda descansar y comer.
✓ Dieta equilibrada y variada de semillas, frutas y verduras frescas.
✓ Jaula limpia y ordenada.
✓ Chequeo anual.

Cobayo

✓ El 70% de su alimentación debe ser heno, 25% verduras y 5% de conejín al día.
✓ Tamaño mínimo de recinto para cada roedor 70cm x 105cm.
✓ Evitar el uso de sustrato para la cama, reemplazar con mantas polares.
✓ Mantener el bebedero con agua fresca.
✓ Limpiar su recinto todos los días.

Perro

✓ Elegir un pienso nutritivo y darles una buena hidratación.
✓ Cepillar diariamente, cortar el pelo y las uñas cuando sea necesario.
✓ Sacar a pasear al menos tres veces al día.
✓ Visitar al veterinario para ponerle las correspondientes vacunas, el chip y desparasitarlo.

Tortuga

✓ Terrario amplio .
✓ Alimentarlo con frutas y verduras.
✓ Buena ventilación.
✓ Temperatura del ambiente menor a 30° y mayor a 23°.
✓ Disposición de bañera.
✓ Sacarlo del terrario para que haga ejercicio.
